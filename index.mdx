## Get started in 4 steps

<Steps>
  <Step title="Install MONAI">
    Install the framework with your preferred package manager:
    ```bash
    pip install monai
    ```
  </Step>
  <Step title="Load and transform medical images">
    Use MONAI's domain-specific transforms to load and preprocess medical images:
    ```python
    from monai.transforms import Compose, LoadImaged, EnsureChannelFirstd, ScaleIntensityRanged

    transforms = Compose([
        LoadImaged(keys=["image", "label"]),
        EnsureChannelFirstd(keys=["image", "label"]),
        ScaleIntensityRanged(keys=["image"], a_min=-57, a_max=164, b_min=0.0, b_max=1.0, clip=True),
    ])
    ```
  </Step>
  <Step title="Build a network and loss function">
    Pick from MONAI's library of medical imaging architectures:
    ```python
    from monai.networks.nets import UNet
    from monai.losses import DiceCELoss

    model = UNet(
        spatial_dims=3,
        in_channels=1,
        out_channels=2,
        channels=(16, 32, 64, 128, 256),
        strides=(2, 2, 2, 2),
    )
    loss_fn = DiceCELoss(to_onehot_y=True, softmax=True)
    ```
  </Step>
  <Step title="Run sliding window inference">
    Efficiently infer on large 3D volumes:
    ```python
    from monai.inferers import SlidingWindowInferer

    inferer = SlidingWindowInferer(roi_size=(96, 96, 96), sw_batch_size=4, overlap=0.5)
    outputs = inferer(inputs, model)
    ```
  </Step>
</Steps>
